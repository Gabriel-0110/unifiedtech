This document defines the **mandatory requirements** for a Microsoft Teams SMS campaign using the **UCAAS Low Volume** use case. These requirements are aligned with Microsoft Telephone Number Services and external SMS aggregator validation criteria.

---

## 1. Scope and Purpose

This SMS program is **service-only** and intended for **1-to-1, conversational messaging** with end users. Marketing, promotional, or bulk messaging is **not permitted**.

Allowed message types:

- Appointment reminders
- Account-related notifications
- Service status updates

Disallowed content:

- Marketing or promotional messages
- Discounts, offers, or advertising
- Automated or bulk campaign messaging

---

## 2. Website Opt-in Requirements

**Opt-in method:** Website only

The opt-in page must clearly demonstrate how users consent to receive SMS messages.

### Required elements on the opt-in page

- A visible **phone number input field**
- An **unchecked consent checkbox**
- Disclosure text displayed **adjacent to the checkbox**

### Required disclosure text

```
By submitting this form, you consent to receive SMS messages from UnifiedTech Solutions by G&G at the phone number you provide. Messages may include service reminders and account-related notifications. Message frequency varies. Msg & data rates may apply. Reply STOP to opt out or HELP for help. Consent is not a condition of purchase.
```

### Required checkbox label

```
I agree to receive SMS messages as described above.
```

---

## 3. Terms and Conditions Requirements

The Terms and Conditions page must include a **dedicated SMS section** that explicitly describes the opt-in process and message usage.

### Required section title

**SMS Messaging Terms (Microsoft Teams)**

### Required Terms and Conditions text

```
UnifiedTech Solutions by G&G offers SMS messaging for service-related communications only. Users opt in by submitting their mobile phone number through our website contact form at https://ggunifiedtech.com/contact and providing explicit consent via an unchecked opt-in checkbox.

SMS messages may include appointment reminders and account-related notifications. Message frequency varies. Message and data rates may apply. Users may opt out at any time by replying STOP or request assistance by replying HELP.
```

---

## 4. Privacy Policy Requirements

The Privacy Policy must explicitly state how mobile phone numbers are used and protected.

### Required Privacy Policy statement

```
Mobile phone numbers collected for SMS communications are used solely for service-related messaging and are not shared with third parties for marketing or promotional purposes.
```

---

## 5. Campaign Description (Teams Admin Center)

The campaign description must clearly define the service-only nature of the SMS program.

### Required campaign description text

```
UnifiedTech Solutions by G&G uses SMS messaging to support clients with service-related communications, including appointment reminders and account notifications. This campaign is intended for 1-to-1, conversational messaging only and does not include marketing or promotional content.
```

---

## 6. Call to Action (Teams Campaign)

The call to action must explicitly describe how users opt in.

### Required call to action text

```
Users opt in by submitting their mobile phone number through our website contact form at https://ggunifiedtech.com/contact and providing explicit consent via an opt-in checkbox to receive service-related SMS messages.
```

## 7. Message Template Requirements

Only service-related message templates are permitted.

### Opt-in confirmation message

```
UnifiedTech Solutions by G&G: You have successfully opted in to receive service-related SMS messages. Msg & data rates may apply. Reply STOP to opt out or HELP for help.
```

### Service reminder message

```
UnifiedTech Solutions by G&G: Reminder that your consultation is scheduled for tomorrow at 10:00 AM. Reply STOP to opt out or HELP for help.
```

### Account notification message (optional)

```
UnifiedTech Solutions by G&G: Your account update has been completed. Reply STOP to opt out or HELP for assistance.
```

---

## 8. Prohibited Practices

The following practices will result in campaign rejection:

- Marketing or promotional SMS content
- Multiple opt-in methods (e.g., SMS keywords, verbal, IVR)
- Pre-checked consent checkboxes
- Bulk or automated messaging
- Fixed message frequency guarantees
